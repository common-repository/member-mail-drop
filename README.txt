Digitize Physical Mail for Your Members. This plugin allows members to manage and access their physical mail digitally.

== Description ==

Member Mail Drop is a WordPress plugin primarily crafted to digitize physical mail for your members, converting it into an easily accessible virtual format. It streamlines email management, enabling bulk sending, delivery tracking, and user email list organization and much more!

**Key Features:**
- Send emails to multiple users
- View a list of all emails sent to users
- Send emails with attachments
- Add and manage email folders
- Allow users to reply to emails
- Notify users on their external emails
- Broadcast messages to individuals
- Scan Documents


For more information, visit the Member Mail Drop website (https://membermaildrop.com/).

== Installation ==

**Automatic Installation**

1. In your WordPress dashboard, go to "Plugins" > "Add New".
2. Search for "Member Mail Drop".
3. Find the "Member Mail Drop" plugin by Hudson Group.
4. Click "Install Now".
5. Once installed, click "Activate".

**Manual Installation**

1. Download the latest version of the Member Mail Drop plugin from the [WordPress Plugin Repository](https://wordpress.org/plugins/member-mail-drop/).
2. Extract the downloaded ZIP file to your computer.
3. Upload the `member-mail-drop` folder to the `/wp-content/plugins/` directory on your web server.
4. Activate the plugin through the 'Plugins' menu in WordPress.
